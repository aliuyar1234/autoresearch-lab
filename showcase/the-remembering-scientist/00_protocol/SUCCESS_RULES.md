# Success Rules - 12h Pilot

## Pilot Success

The bounded pilot counts as successful if all of the following are true:

- baseline noise is measured and documented
- both official arms complete under the frozen 12h budget
- the remembering arm wins on at least 2 of these 3 axes:
  - better confirmed primary result
  - faster time to first promoted keep
  - lower repeated-dead-end rate
- at least one believable memory-driven example exists
- the morning report comparison is visibly more coherent for the remembering arm

## Pilot Inconclusive

The bounded pilot is inconclusive if any of these hold:

- score deltas are tiny relative to baseline noise
- confirm results are effectively random
- there is no clear evidence that historical memory changed proposal choice
- the only visible difference is report formatting

## Pilot Failure

The pilot should be considered failed or blocked if:

- memory seed is too thin to matter
- proposal lineage or memory evidence is not traceable
- one arm is accidentally advantaged by extra context or manual steering
- the campaign does not move enough to support the story

## What The Pilot Must Produce

Minimum required artifacts:

- baseline noise note
- one morning report from each arm
- one compareable hero curve
- one confirm table
- one clean baseline replay
- one clean finalist replay
- one written note explaining whether the pilot should be expanded into the full flagship
